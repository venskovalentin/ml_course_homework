# ML Course Homework — FAMCS BSU 2026

Домашние задания по курсу машинного обучения факультета прикладной математики и информатики БГУ (**ML FAMCS 2026**).

Репозиторий охватывает полный путь от первичной настройки инструментов и разведочного анализа данных до классических алгоритмов классификации, ансамблевых методов, интерпретации моделей и обучения без учителя.

## Содержание

| HW | Тема |
| --- | --- |
| `hw01_setup_tools` | Настройка окружения и инструментов |
| `hw02_EDA` | Разведочный анализ данных (EDA) |
| `hw03_KNN` | Метод k ближайших соседей |

| `hw07_SVM` | Метод опорных векторов |

| `hw09_GradBoost` | Градиентный бустинг |
| `hw10_hp_interpr` | Подбор гиперпараметров и интерпретация моделей |
| `hw11_clastering` | Кластеризация |
| `hw12_pca` | Метод главных компонент (PCA) и снижение размерности |

## Изученные технологии и методы

**Языки и окружение**
- Python, Jupyter Notebook
- управление зависимостями через `requirements.txt`, виртуальные окружения, Git/GitHub

**Работа с данными и визуализация**
- pandas, NumPy
- разведочный анализ данных (EDA), очистка и предобработка
- matplotlib, seaborn для визуализации распределений и зависимостей

**Классические алгоритмы машинного обучения**
- линейная и логистическая регрессия, регуляризация (Ridge, Lasso, ElasticNet)
- метод k ближайших соседей (KNN)
- наивный байесовский классификатор
- метод опорных векторов (SVM), ядровые методы
- деревья решений

**Ансамблевые методы**
- случайный лес (Random Forest), бэггинг
- градиентный бустинг (XGBoost, LightGBM, CatBoost)

**Обучение без учителя**
- кластеризация (K-Means, иерархическая кластеризация, DBSCAN)
- снижение размерности: метод главных компонент (PCA), t-SNE

**Оценка, настройка и интерпретация моделей**
- кросс-валидация, метрики классификации и регрессии
- подбор гиперпараметров (GridSearchCV, RandomizedSearchCV, Байесовская оптимизация)
- интерпретируемость моделей: SHAP, permutation importance, partial dependence (PDP), ICE-кривые

**Стек инструментов**
- scikit-learn как основной фреймворк ML
- optuna, numpy, shap, XGboost
  
## Структура

Каждое задание оформлено в отдельной директории `hwXX_*` и содержит соответствующий Jupyter-ноутбук с решением.
